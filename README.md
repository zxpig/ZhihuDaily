# ZhihuDaily
第三方知乎日报客户端，全部数据来自知乎（Zhihu）API
### 功能介绍
1. 查看日报最新/热门文章：获取最新/热门文章，以及每篇文章的额外信息，如：评论数、收藏数、点赞数。
2. 栏目：获取日报全部栏目和文章。
3. 查看评论：内容分享，文字复制。
4. 文章推送：日报有新文章发布，会给用户发送一条通知。
